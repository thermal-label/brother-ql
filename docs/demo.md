# Live Demo

Type a label below and see a live 1bpp bitmap preview instantly. Connect a Brother QL printer via WebUSB (Chrome or Edge only) to print it directly from your browser.

<LiveDemo />

## Notes

- Preview works in all browsers — no WebUSB required
- Printing requires Chrome 61+ or Edge 79+ in a secure context (`https://` or `localhost`)
- If your printer is not detected, check that [Editor Lite mode is disabled](/hardware#editor-lite-mode)
- Two-color printing requires a QL-800, QL-810W, or QL-820NWB with DK-22251 labels
